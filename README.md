# 🛒 E‑Commerce Website

A modern e‑commerce web application built to simulate online shopping experiences.  
This project demonstrates frontend development, API integration, and responsive design.

---

## 📖 Overview
The website allows users to browse products, view details, add items to the cart, and proceed to checkout.  
It is designed with scalability in mind, making it easy to extend with backend services.

---

## ✨ Features
- Responsive layout with margins and spacing similar to Flipkart/Amazon
- Product listing page with filters and search
- Product detail page with reviews and ratings
- Shopping cart with quantity updates and total price calculation
- Checkout flow with address and payment (dummy integration)
- Wishlist functionality
- Dark/Light mode toggle

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript (ES6), React (optional)  
- **Styling:** TailwindCSS / Bootstrap  
- **Data Handling:** JSON / API integration  
- **Tools:** VS Code, Git, GitHub
